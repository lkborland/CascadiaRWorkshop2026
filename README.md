# CascadiaRWorkshop2026
Repository for materials for Cascadia R Conference 2026 - introduction to R workshop

## Install R and RStudio
Download both R (cran.r-project.org) and RStudio (posit.co/downloads/). 
You can also use the cloud version for free - https://posit.cloud/plans/free. 
Go to https://rstudio-education.github.io/hopr/starting.html for more information.
